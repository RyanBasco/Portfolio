Thesis Project

Guimaras Tourism App

A cross-platform mobile application designed to revolutionize tourist engagement and cultural preservation. Built with Flutter and Firebase, this solution enables real-time tourism recording, cloud synchronization, and seamless data management for tourists exploring the beautiful island of Guimaras.

The app provides an intuitive interface for capturing travel experiences, managing multimedia content, and accessing curated tourism information with real-time updates from the cloud backend.

Technology Stack

Frontend Framework

Dart & Flutter

Cross-platform mobile development for iOS and Android with native performance

Backend & Cloud Services

Firebase

Real-time database, cloud storage, authentication, and synchronization

Key Features

Multimedia Recording

Capture high-quality photos and videos directly from the app for authentic travel memories.

Real-Time Cloud Sync

Automatic synchronization of all data to Firebase for secure backup and multi-device access.

Location Tagging

Geotag your tourism experiences with precise location data and travel history tracking.

Offline Capability

Full functionality even without internet connection - data syncs automatically when online.

Secure Authentication

Firebase authentication ensures only authorized users can access their personal data and recordings.

Cross-Platform

Seamless experience across iOS and Android devices with consistent performance and UI.
